# VF FREE VPN

Одностраничный сайт с 6 бесплатными VLESS+Reality подписками для INCY и Happ. Безлимитный трафик, без ограничений, без регистрации по email — только Telegram.

## Возможности

- 6 VLESS подписок (WVFSTANDART, WVFMINI, WVFWL, WVFBYWILDTASK, WVFWILDVFROBOT, WVFBLACK)
- Копирование подписки в буфер обмена одним кликом
- Вход через Telegram (или вручную по username)
- Проверка подписки на канал @wildVF
- Игра «Орёл и Решка» с выигрышем монет
- Ежедневный бонус (+50 монет каждый день)
- Реферальная система (+50 монет за друга)
- Профиль с аватаркой, статистикой, балансом
- Загрузка своей аватарки
- 3D tilt на карточках подписок
- Плавные анимации, стеклянный дизайн (glassmorphism)
- Конфетти при победе в игре
- Админ-панель (5 кликов по лого): просмотр пользователей, блокировка, настройка бота

## Требования

- Любой статический хостинг (GitHub Pages, Netlify, Vercel)
- HTML5, CSS3, JavaScript (ES6+) — всё на клиенте
- Без сервера, без базы данных (всё в localStorage)
- Для проверки подписки через Telegram: Cloudflare Worker + BotFather
- Сайт работает и без бота — ручной ввод username
